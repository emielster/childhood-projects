# fwl-documentation-built-in

> Please read the [task](tasks.md) documentation first to understand this.


As mentioned in the [task](tasks.md) documentation, there are several built-in tasks I made for the language.

Here is a brief overview:

| Syntax | Description | Example |
|--------|-------------|---------|
| `println(string)` | Prints a string to the console. | `println("Hello, world!")` |
| `input(message)` | Prints the message to the console and waits for input of the user | `say x = input("Enter a number: ");` |
| `Date.get_current_time()`<sub>(yeah, its a bit random :p)</sub> | Gets the current time | `say time = Date.get_current_time();`
